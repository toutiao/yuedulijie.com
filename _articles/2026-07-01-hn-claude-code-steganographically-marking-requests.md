## 原文概要

Hacker News 上的热门帖子标题为“Claude Code is steganographically marking requests”，暗示 Anthropic 公司的 Claude AI 模型在生成代码时可能采用了隐写术（`steganography`）技术，对请求进行标记。这种做法意味着在代码输出中嵌入了不易察觉的隐藏信息，可能用于追踪代码来源、验证模型归属或实现其他形式的数字水印。此话题在 HN 热门榜 (/best) 上引起了关注。
